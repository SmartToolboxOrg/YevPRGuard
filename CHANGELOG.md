# Changelog

All notable changes to **AI Review** are documented here.

## [1.0.0] - 2026-03-20

### Features

- Initial release
- AI-powered code review with inline annotations and quick-fix suggestions (Alt+Enter)
- Full-diff and single-file review modes
- Manual review comment support via right-click context menu
- Publish findings to GitHub PRs with one click
- AI-generated PR title and description from current diff
- AI-generated commit message from staged changes
